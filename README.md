# quiz

**Описание проекта:**

Интерактивная веб-платформа для проведения викторин, которая позволяет пользователям проходить тематические тесты в режиме реального времени.

**Цель проекта:**

Разработка интерактивной платформы для проведения викторин, позволяющей пользователям проходить тестирование с автоматической проверкой ответов в реальном времени.
Платформа включает в себя интеграцию таких функциональных модулей, как система регистрации участников через сессии, динамическая выдача вопросов из базы данных, строгая последовательность шагов (без рандомизации), а также модуль подсчета и вывода итоговых результатов.
В процессе разработки применены ключевые аспекты работы с веб-фреймворком Django.

| Адрес | Описание |
|-------------|-------------|
| 127.0.0.1   | Главная страница   |
| 127.0.0.1/admin/   | Панель администратора  |

Чтобы задать вопросы и ответы к ним, нужно создать суперпользователя, перейти в панель администратора, после чего перейти в MAIN/Tasks.

## Установка:

Если Python не установлен, скачайте и установите его с [официального сайта](https://www.python.org/downloads/).

### Системные требования

- **Версия Python**: 3.9 или выше
- **Операционная система**: Windows / macOS / Linux

Клонировать репозиторий и перейти в него в командной строке:
```python
git clone git@github.com:qqflyflyin/guiz.git
```
```python
cd guiz
```
Установить зависимости из файла requirements.txt:
```python
python3 -m pip install --upgrade pip
```
```python
pip install -r requirements.txt
```
Выполнить миграции:
```python
python3 manage.py migrate
```
Создать суперпользователя (если необходимо):
```python
python3 manage.py createsuperuser
```
Запустить проект:
```python
python3 manage.py runserver
```
